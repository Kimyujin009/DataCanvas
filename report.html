DataCanvas

DataCanvas is a lightweight Windows utility for fast CSV visualization and first-pass regression analysis. It helps students and researchers inspect experiment data quickly without writing code.

The app loads a CSV file, detects numeric columns, draws a scatter plot, computes a linear regression line, shows the regression equation and R-squared, and previews the loaded table in one screen.

1. Planning

In lab classes, assignments, and small experiments, data is often saved as CSV after using Excel or measurement equipment. However, turning that data into a chart and checking the trend line still takes repeated manual work. DataCanvas was planned as a small analysis assistant that reduces this repetitive step and helps users understand their data immediately.

The project goal was not to build a full statistical platform, but to make a stable and easy utility that works within a short contest development time and is still useful in real coursework or research preparation.

2. Components

CSV Loader

Loads user CSV files and built-in sample datasets.

Column Selection

Detects numeric columns automatically and lets the user choose X and Y.

Visualization

Draws a scatter plot and overlays a linear regression line.

Result Panel

Shows data count, regression equation, and R-squared in the same window.

3. Implementation

The app was implemented with Python, PySide6, pandas, matplotlib, and scikit-learn. The UI uses a three-column desktop layout: input controls on the left, plot and table preview in the center, and regression results on the right.

During development, AI coding assistance was used to rapidly iterate on the app structure, error handling, layout cleanup, sample data preparation, and submission document drafting. The workflow was: define the problem, narrow the MVP scope, generate code, test, fix packaging paths, and refine the final submission artifacts.

4. Results

The finished utility can load a CSV file, auto-fill available numeric columns, automatically draw an initial plot from default columns, and show regression information immediately. This makes it suitable for quick experiment review, report preparation, and trend checking before creating presentation materials.

Main screen of DataCanvas showing sample dataset loading, plot rendering, data preview, and regression summary in one view.

5. Strengths

  • Fast first-pass analysis for CSV-based experiment data.
  • Single-screen layout makes the workflow easy to understand in a short demo.
  • Built-in sample datasets help the app demonstrate reliably even without external files.
  • Offline execution is possible after dependency installation or release packaging.

6. Limitations

  • Only linear regression is included in the current MVP.
  • The app focuses on numeric CSV data and does not support advanced spreadsheet features such as multi-sheet Excel analysis.
  • Statistical testing, outlier analysis, and model comparison are intentionally excluded to keep the tool stable and simple.
